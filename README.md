# AI Nutrition Planner 🥗💪

## Overview

AI Nutrition Planner is a personalized nutrition recommendation system that generates daily dietary suggestions based on a user's gender, workout activity, and calories burned.

The goal of this system is to help individuals make better nutrition decisions by adapting dietary recommendations according to their physical activity and energy expenditure.

---

## Problem Statement

Many fitness and nutrition applications provide static diet plans that do not adapt to daily physical activity or calorie expenditure. This can result in inaccurate nutrition recommendations that do not support recovery, performance, or health goals.

There is a need for an intelligent system that dynamically adjusts nutrition recommendations based on a user's daily activity levels.

---

## Solution

AI Nutrition Planner analyzes user information such as gender, workout type, and calories burned to generate personalized nutrition recommendations. The system adjusts calorie intake and macronutrient distribution to support energy balance and recovery.

---

## Features

* Personalized nutrition recommendations
* Calorie adjustment based on workout activity
* Macronutrient distribution suggestions
* Simple and interactive user interface

---

## Tech Stack

* Python
* Machine Learning
* Streamlit
* SQL (for storing user data)

---

## System Workflow

1. User enters personal details and workout activity.
2. System calculates energy expenditure based on calories burned.
3. Machine learning  adjusts daily calorie intake.
4. The system generates personalized nutrition recommendations.
